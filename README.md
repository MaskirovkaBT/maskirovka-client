# Maskirovka Client

TUI-клиент для просмотра юнитов BattleTech. Построен на фреймворке Textual.

## Описание

**Maskirovka Client** — это текстовый пользовательский интерфейс (TUI) для работы с каталогом боевых машин вселенной BattleTech. Приложение предоставляет удобный интерфейс для просмотра юнитов, фильтрации и сортировки с клавиатурной навигацией.

## ⚠️ Дисклеймер

**BattleTech** — зарегистрированная торговая марка [Catalyst Game Labs](https://www.catalystgamelabs.com) / Topps.

## Функциональность

- 📚 **Просмотр справочников**: списки эр и фракций
- 🔍 **Фильтрация юнитов** по множеству параметров:
  - Тип и название юнита
  - Характеристики (PV, размер, броня, структура и др.)
  - Оружейные показатели (short, medium, long, extreme)
  - Особые способности (specials)
- 📊 **Сортировка** по различным полям (название, PV, роль, характеристики)
- 📄 **Пагинация** результатов
- ⌨️ **Клавиатурная навигация**: полное управление без мыши

## Стек технологий

- **Textual** — современный TUI фреймворк для Python
- **httpx** — асинхронный HTTP клиент
- **Pydantic + pydantic-settings** — валидация данных и настроек
- **Python 3.9+**

## Установка и запуск

### Требования

- Python 3.9+
- Ссылка на сервер с данными о юнитах
- Установленные зависимости из `requirements.txt`

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Конфигурация

Создайте файл `.env` в корне проекта (или рядом с .exe):

```
API_BASE_URL=http://127.0.0.1:8000
```

### Запуск приложения

```bash
textual run maskirovka.py
```

## Интерфейс и навигация

### Горячие клавиши

| Клавиша             | Действие |
|---------------------|----------|
| `Tab` / `Shift+Tab` | Переключение между блоками (эры → фракции → таблица) |
| `↑` / `↓`           | Навигация по спискам |
| `Space`             | Выбор/снятие выбора фракции |
| `Enter`             | Открыть детали юнита |
| `Ctrl+s`            | Поиск (загрузить юниты по выбранным критериям) |
| `Ctrl+o`            | Открыть окно сортировки |
| `Ctrl+f`            | Открыть окно фильтрации |
| `Ctrl+←` / `Ctrl+→` | Предыдущая / следующая страница |
| `q`                 | Выход |
| `Escape`            | Закрыть модальное окно |

### Структура интерфейса

```
+----------------------------------+------------------+
| Header (clock, icon)                                |
+----------------------------------+------------------+
| Эры (RadioSet)                   | Фракции          |
| [ ] Succession Wars              | [ ] Draconis     |
| [*] Clan Invasion                | [x] Federated    |
| [ ] Jihad                        | [ ] Lyran        |
+----------------------------------|                  |
| Таблица юнитов                   |                  |
| Название | Роль | PV | MV | ...  |                  |
+----------------------------------+------------------+
| Страница: 1 из 10                 |
+-----------------------------------------------------+
| Footer (горячие клавиши)                            |
+-----------------------------------------------------+
```

### Фильтрация

Окно фильтрации (`Ctrl+f`) позволяет задать:

| Параметр | Описание |
|----------|----------|
| `title` | Поиск по названию (частичное совпадение) |
| `unit_type` | Тип юнита (Mech, Vehicle и др.) |
| `role` | Роль (Brawler, Scout и др.) |
| `specials` | Особые способности (режимы: OR/AND) |
| Числовые | PV, SZ, Short, Medium, Long, Extreme, OV, Armor, Struc, Threshold, MV |

Режимы сравнения для чисел: `eq` (равно), `gt` (>), `gte` (≥), `lt` (<), `lte` (≤)

### Сортировка

Окно сортировки (`Ctrl+o`) позволяет сортировать по:
- `title` — название
- `pv` — Point Value (стоимость)
- `role` — роль
- `short`, `medium`, `long` — оружейные показатели
- `armor`, `struc` — броня и структура
- `mv` — движение

Порядок: `asc` (по возрастанию) / `desc` (по убыванию)
